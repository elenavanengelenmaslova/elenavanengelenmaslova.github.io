### Recent Projects

#### <img src="/assets/images/logos/azl-logo.webp" alt="AZL logo" style="height: 30px; width: auto; vertical-align: middle; margin-right: 8px;">AZL (NN Group) — Pension Integration Platform
**Role:** Lead Software Engineer / Kotlin Tech Lead · 2023–Present

Designed and delivered a greenfield event-driven integration system on Azure to support the Dutch New Pension Law. The platform replaced a legacy on-premise Oracle system and now handles secure data exchange across financial institutions, government bodies, and internal systems — with mTLS, OAuth2, and Sivi-standard messaging built in.

**Technologies:** Kotlin, Azure Functions, Azure Service Bus, Event Hubs, Cosmos DB, Key Vault, Spring Boot, Spring Cloud Function, OAuth2, mTLS, Azure DevOps

#### <img src="/assets/images/logos/PostNL-logo.png" alt="PostNL logo" style="height: 30px; width: auto; vertical-align: middle; margin-right: 8px;">PostNL — Automated Parcel Processing
**Role:** Senior Solution Software Engineer (Kotlin) · 2021–2022

Built an event-driven serverless system on AWS that automated parcel processing and resource allocation using real-time IoT data. The platform enabled smarter route planning and reduced resource waste across PostNL's delivery network.

**Technologies:** Kotlin, AWS Lambda, EventBridge, S3, DynamoDB, SNS, SQS, AppSync, CloudFront, Location Services, AWS CDK
